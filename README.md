# vivid-wavetable

Extracted package containing the `WavetableSynth` operator.

## Local install

```bash
./build/vivid link ./packages/vivid-wavetable
./build/vivid rebuild vivid-wavetable
```

## Package contents

- `src/wavetable_synth.cpp`
- `factory_presets/wavetable_synth.json`
- `vivid-package.json`
